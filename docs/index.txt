CC Network Documentation
========================

This documentation describes the software which powers the `Creative
Commons Network`_.

CC Network implements a simple copyright registry and supports `ccREL
<http://wiki.creativecommons.org/ccREL>`_/`RDFa
<http://wiki.creativecommons.org/RDFa>`_ based discovery of work
information.
