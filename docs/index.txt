Commoner Documentation
====================================

Commoner is an application which implements a simple copyright registry and
supports `ccREL <http://wiki.creativecommons.org/ccREL>`_/`RDFa
<http://wiki.creativecommons.org/RDFa>`_ based discovery of work information.
